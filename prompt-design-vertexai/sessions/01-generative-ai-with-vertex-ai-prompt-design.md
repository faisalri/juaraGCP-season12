
# Session 01 — Generative AI with Vertex AI: Prompt Design

## Objectives
- Understand prompt design basics & patterns.
- Practice writing prompts with role, task, and constraints.

## Steps / Notes
1. Define **goal** and **output format** (JSON/table/bullets).
2. Add **context** (who/what/where) and **tone**.
3. Provide **examples** (few‑shot) when needed.
4. Add **guardrails**: word limits, taboo list, safety hints.
5. Iterate: test with small golden set.

## Prompt Patterns (cheat)
- Role + Task + Constraints
- Few‑shot with labeled inputs/outputs
- Chain‑of‑Thought hints (concise, non‑revealing)
- Critic/Refiner loop
