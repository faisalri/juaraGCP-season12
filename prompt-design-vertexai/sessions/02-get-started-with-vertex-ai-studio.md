
# Session 02 — Get Started with Vertex AI Studio

## Objectives
- Create a prompt in Vertex AI Studio and evaluate outputs.
- Save prompt templates and export for SDK usage.

## Steps / Notes
1. Open **Vertex AI Studio** → Text Generation.
2. Choose a base model → start with **Gemini** family.
3. Add system instructions, input variables, and output format.
4. Use **Test** panel with multiple examples.
5. Save **Prompt Template** for reuse.

## Tips
- Keep **temperature** low for determinism when evaluating.
- Use **safety** settings aligned to enterprise needs.
