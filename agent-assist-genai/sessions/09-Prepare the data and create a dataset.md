# Prepare the data and create a dataset

## Objectives
In this lesson, you'll learn about the essential process of preparing your conversational data and ingesting it into an Agent Assist dataset. The video below covers critical steps for storing your raw data in a Google Cloud Storage (GCS) Bucket, detailing how to create a new bucket if needed, and emphasizing the precise JSON format required for optimal model training. You'll also learn how to upload your validated data and, finally, import it into the Agent Assist Console to create a new dataset, transforming your raw transcripts into a format ready for Smart Reply model training.
